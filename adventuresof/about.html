About our game The Adventures Of...

The Adventures Of... is written in Java using the LIBGDX game engine. it is comprised of 2D textures and sprites rendered on an orthographic view to give it the retro feel, Due to the use of LIBGDX, The Adventures Of... is available in HTML, IOS and desktop.

The map was created using a 3rd party tool called Tiled, creating maps using a mixture of custom textures and 3rd party tilesets from online game asset libraries and tutorials. LIBGDX provides the necessary libraries to interact with the Tiled map.
