About our game The Adventures Of...

The Adventures Of...[Insert Name Here] is written in Java using the LIBGDX game engine. It is comprised of 2D textures and sprites rendered on an orthographic view to give it the retro feel. Due to the use of LIBGDX, The Adventures Of...[Insert Name Here] is available in HTML, IOS and desktop.

For creating the map, we will utilise a 3rd party tool called Tiled, creating maps using a mixture of custom textures and 3rd party tilesets from online game asset libraries and tutorials. LIBGDX provides the necessary libraries to interact with the Tiled map.


Due to the use of LIBGDX, our solution is split into four projects, the core project - in which we will add code that is common to all platform versions of the game, and three more projects representing each the HTML, IOS and desktop version of the game - in which we will store our platform specific code.
